 Avaliação Prática: Primeiros Passos no Frontend

## 🎯 Objetivo
Desenvolver uma página web simples utilizando os conceitos fundamentais de **HTML** e **CSS** apresentados em aula. O foco desta avaliação é a correta estruturação do documento e a aplicação de estilos básicos.

---

## 📋 Requisitos do Projeto

### 1. Estrutura do Documento (HTML)
O aluno deve criar um arquivo chamado `index.html` contendo a estrutura essencial de um documento web:
- Declaração do tipo de documento: `<!DOCTYPE html>`.
- Tag raiz: `<html>`.
- Cabeçalho técnico: `<head>` contendo o `<title>` da página.
- Corpo da página: `<body>` onde todo o conteúdo visível será inserido.

### 2. Marcação de Conteúdo
A página deve conter, no mínimo, os seguintes elementos:
- **Títulos:** Uso de `<h1>` para o título principal.
- **Parágrafos:** Uso de `<p>` para descrições de texto.
- **Listas:** Pelo menos uma lista organizada (`<ol>`) ou não organizada (`<ul>`).
- **Links:** Inserção de pelo menos um link externo utilizando a tag `<a>`.
- **Imagens:** Inclusão de uma imagem utilizando a tag `<img>` com os atributos necessários.

### 3. Estilização (CSS)
Crie um arquivo `style.css` e vincule-o ao seu HTML. A estilização deve contemplar:
- **Seletores:** Uso de seletores de **Tag**, **Classe** (`.minha-classe`) e **ID** (`#meu-id`).
- **Cores:** Aplicação de cores em pelo menos dois formatos diferentes:
  - Nomes de cores (ex: `blue`).
  - Código Hexadecimal (ex: `#FFFFFF`).
  - Valores RGB ou RGBA.
- **Formatação de Texto:**
  - Definição de tamanho de fonte (`font-size`).
  - Alinhamento de texto (`text-align`).
  - Definição de família de fonte (`font-family`).
- **Bordas e Fundos:** Uso de `background-color` e `border`.

---

## 🛠️ Instruções de Execução
1. Crie uma pasta para o seu projeto no seu computador.
2. Desenvolva o código HTML focando na semântica básica.
3. Aplique o CSS para dar identidade visual à página, respeitando o fluxo normal dos elementos (sem uso de layouts complexos como Flexbox).
4. O projeto deve ser visualizado corretamente em qualquer navegador moderno, que interpretará as tags e estilos para exibir a página [6].

---

## ✅ Critérios de Avaliação
- [ ] O código HTML está bem estruturado e identado?
- [ ] As tags foram fechadas corretamente (exceto as que não possuem fechamento como `<img>`)?
- [ ] Os seletores CSS estão sendo aplicados corretamente aos elementos?
- [ ] As propriedades de texto e cores foram utilizadas conforme solicitado?
